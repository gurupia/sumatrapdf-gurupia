---
description: How to test ANSI Codepage Support
---

1. Open a text file or HTML file with a specific encoding (e.g. Windows-1251).
2. Go to `Settings` -> `Encoding`.
3. Verify that the current encoding is checked.
4. Select a different encoding from the list.
5. Verify that the document reloads and displays correctly with the new encoding.
6. Check the status bar (bottom right or toolbar info area) to see if the encoding name is displayed.
